Recent Projects

Objective-based Reinforcement Learning for Robotics

A practical way to train in simulation and deploy learned policies on robots. Tested on the Kinova robotic arm.

Check out the Github repository.

Complex Cobotics

An interface for an interactive and collaborative robot. This project is a unique framework for its goal, bringing together a few different ideas - a cognitive architecture, a collaborative/rule-following component, and automatic plan-based motor control. The effectiveness has yet to seriously be measured, but it's a start.

More in-depth descriptions are on the Github repository.

LLM Planning Pruner

I worked with a team of three other graduate students to demonstrate a novel usage of LLMs, namely, using them in a cognitive architecture to prune planning domains of complex environments. I took a leading role in the initial development by laying the foundation of the implementation for the rest of the group to add. See Github for the final submission.

Ned2Autonomy

An application of a brain-computer interface (a functional Near Infrared Spectroscopy (fNIRS) machine) to decide control over a robotic arm (remote or autonomous).

Again, the repository.
